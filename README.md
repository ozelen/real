# Isomorphic ES6 React application boilerplate

Using Babel 6.3, React 0.14, React Router 2.0, Express 4.13, JADE, Browserify 13
Fully NPM scripts driven (no extra task runners)

## Usage:

``` sh
$ npm start
```
